# Начало работы

## Введение в NarrativeCraft

**Что такое NarrativeCraft?**

NarrativeCraft — это мод для Minecraft, который превращает игру в пространство для создания интерактивных историй с выбором и разветвленными повествованиями, реагирующими на решения игрока.

В основном он предназначен для создания игры в игре и во многом вдохновлен проектами, которые мне нравятся, такими как Life Is Strange и Until Then.

Он представляет собой комплексное решение (запись, кат-сцены, рендеринг диалогов, управление сюжетом) и прост в использовании.

**Как работает этот мод?**

NarrativeCraft существует благодаря проекту с открытым исходным кодом [Ink](https://github.com/inkle/ink), язык повествовательных сценариев, созданный компанией [Инкль](https://www.inklestudios.com/) для написания интерактивных историй.
Он разработан таким образом, чтобы быть простым в освоении и не требовать написания кода, предлагая при этом достаточно мощных функций для расширенного структурирования.

Адаптация Ink на Java была разработана [блейдкодером](https://github.com/bladecoder).

NarrativeCraft основывается на работе Ink и bladecoder, делая возможным интерактивное повествование в Minecraft.

## Предпосылки

**Базовые знания Ink**

У Ink есть своя официальный [документация](https://github.com/inkle/ink/blob/master/Documentation/WritingWithInk.md).

Первые 5 разделов дадут вам базовое представление о том, как работает Ink, поэтому вы сможете легко следовать документации NarrativeCraft.

**Inky**
Чтобы написать создать историю и заняться написанием сценариев, вам нужно установить [Inky](https://github.com/inkle/inky/releases).

Inky — это IDE(Интегрированная среда разработки) с несколькими функциями, которые помогут вам написать историю и управлять несколькими файлами в рамках одного проекта.


# Краткий обзор

Теперь, когда вы знакомы с основами Ink и установили Inky, пришло время узнать, как работает NarrativeCraft.

**Вход в мир**

При присоединении к миру, если NarrativeCraft не был инициализирован, он создаст каталог под названием narrativecraft в папке вашего мира.

Он содержит несколько файлов и папок, давайте разберем их:

*Основы*

- build
Эта папка содержит только один файл с именем *story.json* который вы будете постоянно обновлять. Это построенная история из вашего сценария, использованного для ее реализации.

- chapters
Это список глав с их сценами и данными сцен (анимация, NPC, кат-сцены и т.д.). Никогда не следует удалять, редактировать или создавать в нем новые файлы вручную, в игре все обрабатывается автоматически.

- characters
Список ваших глобальных персонажей.

- data
Разная информация, такая как значения диалогового окна по умолчанию, параметры мира или фон главного экрана.

- saves
Содержит один файл с именем *save.json* в котором есть вся информация для файла сохранения.

** Что вы будете использовать**
- main.ink
Это файл, который вам нужно будет открыть, чтобы написать свою историю. Не открывайте отдельный файл Ink сцены, так как вы не сможете скомпилировать свою историю.
Не пишите в нём; он доступен только для чтения, так как этот файл обрабатывается в игре для автоматической записи *INCLUDE* (список ваших сцен).

- funs.ink
Если вам нужно написать функции, которые вы будете использовать в нескольких файлах, напишите их здесь.

- vars.ink
Переменные, которые будут использоваться на протяжении всей истории, такие как имя или важный выбор.

**Войдите в режим разработки**

Если вы находитесь на главном экране, когда присоединяетесь к миру, и хотите закрыть его, не покидая мир, просто нажмите CTRL LEFT пять раз. Кнопка с надписью Development Environment появится в правом верхнем углу экрана, что позволит вам получить доступ к миру и всем его данным в NarrativeCraft.

**Управление историей**

Чтобы открыть менеджер историй, нажмите N. Откроется экран со списком глав и их сценами, данными сцен и т.д.
Здесь вы можете создавать данные или получать к ним доступ.

**Сессия**

Чтобы легко работать в сцене, записывать или создавать кат-сцену, вам нужно назначить себе сессию.
Сессия — это ваше текущее задание по главе и сцене.

Чтобы установить его, запустите */nc session set %chapter_index% %scene_name%*.

**Построение истории**

Чтобы создать и экспортировать историю Ink для использования в NarrativeCraft:

В Inky перейдите в меню «Файлы».
Нажмите Экспорт в JSON.
Перейдите в папку: narrativecraft/build.
Назовите экспортированный файл: story.json.

*Проверка*

Перед воспроизведением истории он выполнит проверку и проверит каждый тег.
Если тег содержит ошибку, история не будет запущена, и вы получите сообщение об ошибке, указывающее, где находится проблема: индекс главы, название сцены, строка, текст строки и сообщение об ошибке.

Чтобы проверить историю, не проигрывая ее, запустите */nc story validate*.
